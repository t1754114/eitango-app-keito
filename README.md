# えいたんごアプリ

小学生向けの英単語学習Webアプリです。  
カテゴリを選び、英単語カード・クイズ・単語一覧で学習できます。

## 概要

このアプリは、ブラウザだけで動作する静的な英単語学習アプリです。

主な対象は、英語を学び始めた小学生です。  
英単語、日本語訳、例文、日本語訳つき例文、発音確認を組み合わせて、基礎的な英単語を学習できます。

## 主な機能

- カテゴリ別の英単語学習
- 英単語カードによる学習
- 日本語訳を選ぶクイズ
- 単語と例文の音声読み上げ
- カテゴリ別の単語一覧表示
- スマートフォンでも使いやすいシンプルな画面

## 画面構成

### トップ画面

カテゴリを選択し、以下の学習方法を選べます。

- ことばをおぼえる
- クイズをする
- 一覧

### 英単語カード画面

選択したカテゴリの単語を1語ずつ表示します。

表示内容：

- 英単語
- 日本語訳
- 例文
- 例文の日本語訳
- 単語の発音ボタン
- 例文の読み上げボタン

### クイズ画面

表示された英単語に対して、正しい日本語訳を選びます。  
クイズ終了後に結果を表示します。

### 単語一覧画面

選択したカテゴリの英単語と日本語訳を一覧表で確認できます。
